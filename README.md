# SIMOC

## ASU SER-401 Capstone Team 15

SIMOC - A Scalable Model of an Isolated, Off-world Colony


## Running Server

### Normal Run

`python -m simoc_server`

### Debug Mode
`python -m simoc_server --debug`

### Specify Port
`python -m simoc_server --port 9000`

### Run Locally
`python -m simoc_server --run_local`
